Bem-vindo ao projeto de análise de textos

Implementação web de projeto feito no curso de Python da Universidade de São paulo (USP), Utilizando o framework Django. O projeto ainda está em fase de desenvolvimento

Quem pode usar

Qualquer pessoa que deseja analisar um texto próprio e ver se está repetindo muitas palavras e assim melhorar a qualidade do mesmo, seja para estudo ou até mesmo trabalho

Como usar:

O projeto basicamente pega um texto e faz uma análise de quantas palavras foram usadas, quantidade de palavras únicas, quantidade de palavras diferentes, repetição de cada palavra, quantidade de sentenças e quantidade de frases. É só cliclar no botão Analisar Texto

Obs: Frases são trechos de texto separados por uma vírgula (,), dois pontos (:) ou ponto e vírgula (;). Sentença são trechos separados por um ponto, ponto de exclamação (!) ou ponto de interrogação (?).

A funcionalidade de comparar textos e ver a assinatura de um texto logo será adicionada.
